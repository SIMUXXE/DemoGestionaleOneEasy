# Demo Gestionale – Test Tecnico .NET

Questo repository contiene la soluzione realizzata come test tecnico per la valutazione delle competenze di sviluppo .NET, architettura e approccio progettuale.

Il progetto **non è pensato per la produzione**, ma per mostrare:
- organizzazione della soluzione
- separazione delle responsabilità
- uso di tecnologie moderne
- capacità di ragionamento architetturale

## Obiettivo
Realizzare una soluzione unica composta da:
- Backend REST API (.NET moderno, Minimal API)
- Frontend Web (Blazor Server)
- Frontend Desktop (WinForms .NET Framework 4.8)

Tutti i progetti convivono nella stessa solution.

## Tecnologie utilizzate
- .NET 8
- ASP.NET Core Minimal API
- Blazor Server
- WinForms (.NET Framework 4.8)
- Entity Framework Core
- SQL Server
- Microsoft.Data.SqlClient
- Git

Per i dettagli architetturali fare riferimento al file `architecture.md`.
